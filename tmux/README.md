# Tmux

Install plugins

```
prefix + I
```

tmux-sidebar

```
prefix + TAB
prefix + BACKSPACE (focus)
```

tmux-copycat

```
prefix + /
n - next match
N - prev match
```

tmux-urlview

dependencies

```bash
brew install urlview
```

```
prefix + u
```

tmux-open

in visual mode selection highlight filename and press `o`

tmux-fpp (facebook pathpicker)

```
prefix + f
```

tmux-yank

```
prefix + y (copy command line)

selection then y (copy selection)
```
