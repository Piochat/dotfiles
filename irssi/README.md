Commands

```
/msg nickserv register [password] [email]
/msg nickserv identify [password]
/msg nickserv help
/join #channel
/leave
/msg [other person's nickname] [initiating message, like 'hi']
/names
/whois nickname
nickname: ping # get attention
/connect -ssl irc.mozilla.org 6697
```

Keybindings

```
ctrl+p, ctrl-n switch windows

ctrl+x change servers
```
